## Description

Outil de facturation professionnel développé sous Excel avec VBA.
Conçu pour les petites entreprises souhaitant automatiser leur
processus de facturation sans logiciel dédié.

## Fonctionnalités

- Génération automatique de numéros de factures séquentiels
- Calcul automatique HT / TVA 20% / TTC
- Export PDF en un clic
- Historique complet des factures
- Base de données clients intégrée
- Tableau de bord mensuel (CA, nb factures, impayées)
- Sauvegarde rapide et protection des données

## Technologies utilisées

- Microsoft Excel 2016+
- VBA (Visual Basic for Applications)
- Formules Excel : SOMME, AUJOURD'HUI, validation de données

## Installation

1. Télécharge le fichier `FacturationPro.xlsm`
2. Ouvre-le avec Excel (activer les macros si demandé)
3. Commence par l'onglet MENU

## Structure du projet

| Feuille     | Rôle                                   |
|-------------|----------------------------------------|
| MENU        | Navigation principale                  |
| FACTURE     | Formulaire de création de facture      |
| CLIENTS     | Base de données clients                |
| HISTORIQUE  | Archive de toutes les factures         |
| DASHBOARD   | Tableau de bord statistiques mensuels  |
